⑦ Key Prelims Traps — Don't Get Caught!
Q: Which model for 1st Plan?
✅ Harrod-Domar Model
Trap: Don't say Mahalanobis (that's 2nd Plan)
Q: Which model for 2nd Plan?
✅ Mahalanobis Model
Focus: Heavy industries, socialist pattern (IPR 1956)
Q: When was Plan Holiday?
✅ 1966–1969 (3 Annual Plans)
Trap: Green Revolution started during this — not a "dead" period
Q: Rolling Plan — which govt?
✅ Janata Government (1978–80)
Trap: Not Congress — Janata party introduced it
Q: When Planning Commission abolished?
✅ January 1, 2015
Replaced by NITI Aayog — not in 2014 when announced
Q: Agriculture GDP share vs labour?
✅ 16% GDP but 43% labour force
Disguised unemployment — biggest structural issue
Q: Last Five Year Plan?
✅ 12th Plan (2012–17)
After this, NITI Aayog replaced Planning Commission
Q: GST introduced when?
✅ July 1, 2017
Replaced cascading taxes — helped manufacturing

🧠 Plan Models — Quick Recall

1st Plan → Harrod-Domar H for 1st — "H-1" — focus: Agriculture
2nd Plan → Mahalanobis M for 2nd — Macro/industry — 1956 IPR
Rolling Plan → Janata (1978) "Rolling" like revolving govt — unstable period
Plan Holiday → 1966–69 3 wars + drought = 3 annual plans instead
12th Plan → Last (2012–17) After this → NITI Aayog, no fixed plans

The Core Story in One Line
Planned economy (1951) → Heavy industry focus → License Raj bottlenecks → Weak manufacturing → 1991 reforms → IT & Services boom → Service-led economy (54% GDP)

Agriculture still employs 43% but contributes only 16% GDP → disguised unemployment remains India's biggest challenge. Make in India, GST (2017), Labour Code reform (4 codes from 44 laws) are attempts to fix manufacturing.
🌾 Agri → 💻 Services 📋 Harrod-Domar = 1st 🏭 Mahalanobis = 2nd 🌴 Holiday = 1966–69 🔄 Rolling = 1978–80 ⭐ 1991 = Turning Point ✋ Planning Comm. = 2015 📊 Services > 50% GDP
