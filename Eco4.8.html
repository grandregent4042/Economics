The Full Connected Flow
Direct Tax Reform → Better compliance → Higher revenue → Larger divisible pool → Finance Commission distributes (41% vertical + weighted horizontal) → Grants-in-Aid (Art. 275) fill gaps → CSS + CS schemes deliver services on ground. This is cooperative fiscal federalism in practice — Centre collects, Constitution distributes, States deliver.
🖥️ Faceless = Transparent ⚖️ Art. 280 = Finance Commission 📊 41% Vertical · 45% Income Distance 🤝 CSS = Shared · CS = Centre Only 🇮🇳 Cooperative Federalism
