The Core Connecting Idea
A foreign investor can enter India via equity (FDI/FPI) or debt (ECB).
If ownership ≥ 10% → FDI (long-term, active, primary market).
If < 10% → FPI (short-term, passive, secondary market).
If India borrows abroad → ECB (dollar risk on India). If Masala bond → currency risk shifts to foreign investor.
If Indian company goes abroad → ODI (reverse of FDI). To raise capital abroad without direct listing → ADR/GDR.
📊 10% Rule 🏭 FDI = Control 📈 FPI = Portfolio 💼 ECB = Debt 🌶️ Masala = INR Risk Shift ↗️ ODI = India Abroad
